NYC Taxi Data Analysis - Project 2026
Φοιτήτρια: Ιωάννα – Γεωργία Ρίζου

ΑΜ: 2121221

Περιγραφή
Το παρόν project αφορά την ανάλυση δεδομένων των NYC Taxi για το έτος 2024, υλοποιημένη σε κατανεμημένο περιβάλλον Kubernetes (K8s) με τη χρήση της μηχανής Apache Spark. Το αποθετήριο περιλαμβάνει τον κώδικα για την επεξεργασία, την εξαγωγή στατιστικών και την απάντηση στα ερωτήματα Q1, Q2 και Q3, όπως περιγράφονται στην τελική αναφορά.

Περιβάλλον Εκτέλεσης
Η εκτέλεση πραγματοποιήθηκε σε cluster με την ακόλουθη αρχιτεκτονική πόρων ανά ερώτημα:

Driver: 1 Core, 1024 MiB Memory

Executors: 2 Pods (1 Core, 2048 MiB Memory έκαστο)

Σύστημα Αρχείων: HDFS

Δομή του Αποθετηρίου
q1.py, q2.py, q3.py: Πηγαίος κώδικας των ερωτημάτων.

common.py: Κοινές βιβλιοθήκες για SparkSession και διαχείριση μετρικών.

scripts/: Scripts υποβολής (submit_q1.sh, submit_q2.sh, submit_q3.sh) και κεντρικό script εκτέλεσης (run_all.sh).

RUNBOOK.md: Αναλυτικές οδηγίες αναπαραγωγής αποτελεσμάτων.

LLM_USAGE.md: Δήλωση χρήσης εργαλείων LLM.

Οδηγίες Εκτέλεσης
Για την αναπαραγωγή των αποτελεσμάτων, βεβαιωθείτε ότι έχετε ρυθμίσει το περιβάλλον σας (source ~/bigdata-env.sh) και εκτελέστε:

Bash
./scripts/run_all.sh
Περισσότερες λεπτομέρειες στο αρχείο RUNBOOK.md.
